<Section icon={Shield} title="Dungeon Farming and Weekly Maintenance">

While codes provide a massive initial injection of resources, active gameplay is your primary source of sustainable **project mirror labyrinth fragments** over time. Completing dungeon runs and defeating enemies in the maze-like stages will steadily build your fragment reserves.

### Tips for Efficient Dungeon Farming
- **Balance Your Deck**: Do not neglect defense cards. A deck loaded purely with attack cards will quickly fall to high-damage bosses in the deeper floors.
- **Pathing Choices**: When navigating the labyrinth, plan your route to target high-reward encounters that drop fragments and upgrade materials.
- **Utilize the NO ENTRY Gate**: You can easily start a dungeon run by interacting with the physical **NO ENTRY gate** in the lobby or using the TAB menu.

### Scheduled Downtime Alert
Keep in mind that the game undergoes scheduled weekly maintenance to apply patches, balance cards, and rotate code rewards. 

- **Maintenance Day**: Every Tuesday
- **Approximate Time**: 2:00 P.M.
- **Impact**: Server access, dungeon matchmaking, and code redemptions will be temporarily unavailable during this window.

<Callout type="warning" title="Server Desync">
If you attempt to redeem a code immediately following a game update and receive an error, log out of Roblox and rejoin a newly updated server. Older servers often run on outdated code lists.
</Callout>
