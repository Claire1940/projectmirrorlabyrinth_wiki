<Section icon={Swords} title="最強攻撃・防御カード ティアリスト">

**project mirror labyrinth deck**の強さは、挑戦中にドラフトする具体的なカードによって大きく左右されます。カードプールは装備しているOutfitによって変化しますが、現在の2026年メタにおいて一貫して優れたパフォーマンスを発揮する特定のアーキタイプが存在します。以下に、ダンジョン探索中に狙うべき優秀なカードタイプの分類を示します。

| ティア | カードタイプ | 主な効果 | 戦略の焦点 |
| :--- | :--- | :--- | :--- |
| **S-Tier** | **多段攻撃** | 複数回の打撃でダメージを与える | 敵のシールド破壊やトリガーの誘発に最適 |
| **S-Tier** | **ヘビーブロック** | 強力な固定値のダメージ軽減を提供 | ダンジョン終盤を生き抜くために不可欠 |
| **A-Tier** | **状態異常付与** | 出血や火傷などのデバフを適用 | 長期化するボス戦で非常に有効 |
| **B-Tier** | **ユーティリティドロー** | 手札にカードを追加で引き込む | コンボの勢いを維持するのに便利 |

デッキを構築する際は、「3対1の法則」を意識してください。攻撃カード3枚に対して、少なくとも1枚は価値の高い防御カードを確保します。この比率を保つことで、攻撃性と安全性の健全なバランスを維持できます。

<Callout type="warning" title="重要">
カードは特定のパッシブとの相互作用において、大文字・小文字を区別（case-sensitive）します。装備しているOutfitのアビリティ要件と一致しているか、カードの説明を常に注意深く読んで確認してください。
</Callout>

### カードアーキタイプとシナジー

異なるOutfitは、平凡なカードを強力なカードへと変貌させるユニークなボーナスを提供します。例えば、「Defragmenting（再構成）」に焦点を当てたOutfitは、すべての防御カードの威力を高め、「タートル（要塞）」ビルドを極めて実用的なものにします。逆に、**Standard Reflections**や**Targeted Reflections**から入手できる高レアリティのOutfitは、ダメージを蓄積させることで恩恵を得られる、ハイリスク・ハイリターンな攻撃的プレイスタイルを好む傾向があります。

| アーキタイプ | キーカード | 推奨されるOutfitの焦点 |
| :--- | :--- | :--- |
| **アグロ** | 多段ヒット、高クリティカル | Targeted Reflections（攻撃型） |
| **耐久（ストール）** | ヘビーブロック、回復 | Standard Reflections（防御型） |
| **テクニカル** | ドローソース、状態異常 | パッシブ解放済みのDefragmented Outfit |

<Section icon={Target} title="ステップ・バイ・ステップ：最初のデッキ構築">

初めて実用的な**project mirror labyrinth deck**を構築するのは難しく感じるかもしれません。以下の手順に従って、ラビリンスの深層に挑む前に強固な基礎を築きましょう。

<Steps>
  <Step num="1" title="スターター報酬を受け取る">
    ゲームを起動し、すぐに**UrbanPlague**や**Welcome**などの有効なコードを使用します。これにより、最初のReflectionsを引くために必要なFragmentsが手に入ります。
  </Step>
  <Step num="2" title="最高のOutfitを装備する">
    Reflectionsから獲得したOutfitがインベントリにあるか確認します。最もレアリティが高いもの、または好みのプレイスタイル（攻撃型 vs 防御型）に合ったパッシブを持つものを装備しましょう。
  </Step>
  <Step num="3" title="ダンジョンに入る">
    **TAB**キーを押して「Play」を選択するか、ロビーにある**NO ENTRY**ゲートを通り抜けます。初期デッキは、あなたの基礎ステータスとOutfitによって決定されます。
  </Step>
  <Step num="4" title="戦略的にカードをドラフトする">
    戦闘に勝利するたびに、カードの選択肢が提示されます。常にOutfitのパッシブを補完するカードを選んでください。防御型のOutfitを使用している場合は、「ブロック」を付与するカードを探します。
  </Step>
  <Step num="5" title="Defragmentとアップグレード">
    挑戦が終了したら（勝敗に関わらず）、獲得したCondenseとSeeds of Lightを使ってOutfitをDefragment（再構成）します。これにより永続的なパッシブバフが解放され、次の挑戦でデッキがさらに強力になります。
  </Step>
</Steps>

<Callout type="info">
このゲームはローグライトであることを忘れないでください。挑戦に失敗することは敗北ではなく、獲得したリソースを使ってキャラクター、そしてデッキを永続的に強化するチャンスです。
</Callout>

</Section>

<Section icon={Trophy} title="デッキ成長のための通貨とリソース管理">

トップティアの**project mirror labyrinth deck**を維持するためには、複数の異なる通貨を管理する必要があります。各リソースには、新しいカードの獲得（Reflections経由）や、既存カードの強化（Defragmenting経由）といった特定の役割があります。

| リソース | 主な用途 | 入手方法 |
| :--- | :--- | :--- |
| **Fragments** | Reflectionsの利用、スキルの習得 | コード、ダンジョン報酬、マイルストーン |
| **Condense** | OutfitをDefragmentしてパッシブを解放 | 専用コード、高難度ダンジョンのドロップ |
| **Seeds of Light** | 高度なOutfitのアップグレード | イベントコード、レアダンジョンルート |
| **EGO Tickets** | 強力なEGOアイテム用の特別ガチャ | 期間限定コード、コミュニティイベント |

これらのリソースを効果的に管理することが、長期的な成功の鍵となります。**Targeted Reflection**バウチャーが手に入るマイルストーンが近い場合は、手持ちのFragmentsをすべてStandard Reflectionsに費やしてしまわないように注意しましょう。

<Callout type="tip" title="貯蓄戦略">
**SR Guaranteed Mega Tickets**は大型アップデートのために温存しておきましょう。このRobloxタイトルは頻繁にアップデート（例：「[LEI HENG + NOT BINAH]」など）され、より高レアリティの排出が価値を持つ新コンテンツが実装されます。
</Callout>
