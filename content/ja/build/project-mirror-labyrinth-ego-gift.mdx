<Section icon={Target} title="EGOギフトとチケットの入手方法">

適切な**Project Mirror Labyrinth EGOギフト**を入手するには、能動的なゲームプレイと戦略的なリソース交換の両方が必要です。ほとんどのギフトは攻略中に見つかりますが、メタ的な進行（永続的な強化）は、より良いドロップや特化した開始ボーナスを解放するためのEGOチケットと「破片（Fragments）」に依存しています。

EGOギフトのプールに影響を与える主な方法は、鏡像（Reflection）システムです。破片やチケットを使用して衣装（Outfit）を引くことで、特定のギフトカテゴリーとシナジーを持つ固有のパッシブ能力を手にいれることができます。

| 入手方法 | 説明 | 入手頻度 |
| :--- | :--- | :--- |
| **ダンジョンのエンカウント** | エリート敵の撃破後やミステリールームで発見 | 非常に高い |
| **EGOチケット** | 特定のEGO進行に使用される特別なバウチャー | レア |
| **引き換えコード** | 開発者から提供されるコミュニティ報酬 | 定期的 |
| **ショップ交換** | 破片を特定の育成素材と交換 | 普通 |

<Callout type="warning" title="重要">
EGOチケットは、標準的な鏡像チケットよりもはるかに希少です。基本バナーには決して使わず、高ティアのEGO関連パッシブが登場するターゲットローテーションまで待機しましょう。
</Callout>

</Section>

<Section icon={Swords} title="EGOギフトのカテゴリーと優先順位">

すべてのEGOギフトが同じ価値を持つわけではありません。Project Mirror Labyrinthでは、ギフトは一般的に「攻撃型」「防御型」「ユーティリティ型」の3つの主要なアーキタイプに分類されます。現在の衣装の強みや、攻略中のダンジョンの階層に応じて優先順位を切り替える必要があります。

<CardGrid cols={3}>
  <Card title="攻撃型ギフト">
    - **ダメージ倍率**: 攻撃カードの基本威力を増加させます。
    - **状態異常付与**: ヒット時に火傷、出血、振動などを付与します。
    - **クリティカル率**: 高ダメージロールの発生確率を高めます。
  </Card>
  <Card title="防御型ギフト">
    - **防御維持**: 防御値の一部を次のターンに持ち越します。
    - **HP再生**: 戦闘終了ごとに体力の一定割合を回復します。
    - **ダメージ軽減**: 敵の攻撃からのダメージを固定割合で減少させます。
  </Card>
  <Card title="ユーティリティギフト">
    - **カードドロー**: ターンごとに引くカードの枚数を増やします。
    - **リソース獲得**: 追加の破片や凝縮（Condense）を提供します。
    - **コスト減少**: 高ティアカードのエネルギーコストを下げます。
  </Card>
</CardGrid>

ロードアウトを構築する際は、「コア・シナジー」を目指してください。防御的なカウンター攻撃に依存する衣装を使用している場合は、防御を強化したり反射ダメージを与えたりするギフトを優先しましょう。逆に「ガラスの大砲（高火力・低耐久）」ビルドの場合は、敵が反応する前に戦闘を終わらせるために、スピードとカードドローに完全に集中します。

| ギフトティア | 推奨されるフォーカス | 実用的な用途 |
| :--- | :--- | :--- |
| **Sティア** | カードドロー & エネルギー | 戦闘の勢いを維持するために不可欠 |
| **Aティア** | パーセンテージダメージ | 終盤のボスに対するスケーラブルな戦力 |
| **Bティア** | 固定防御値 | 序盤の生存率を高めるのに有用 |
| **Cティア** | リソースバフ | 破片を集めている場合のみ選択 |

<Callout type="info" title="メカニクスノート">
一部のEGOギフトには「デフラグメント（Defragmentation）」の可能性があります。凝縮（Condense）と光の種（Seeds of Light）を消費することで、アカウントの衣装に紐付いた特定のパッシブ能力の基本パワーをアップグレードできます。
</Callout>

<Section icon={Map} title="ダンジョン攻略への統合ステップ">

**Project Mirror Labyrinth EGOギフト**を効果的に使用するには、ダンジョン攻略の流れを理解する必要があります。すでに収集したギフトに基づいてルートを計画するプレイヤーが、より多くの報酬を得られます。

<Steps>
  <Step num="1" title="迷宮に入る">
    Robloxのエクスペリエンスを起動し、TABメニューまたはロビーにある「NO ENTRY」ゲートを使用して攻略を開始します。
  </Step>
  <Step num="2" title="初心者報酬を受け取る">
    **Welcome**などの有効なコードを引き換え、序盤のアップグレードや鏡像（Reflection）のために十分な破片を確保します。
  </Step>
  <Step num="3" title="最初のギフトを見極める">
    最初のエンカウント後、通常はEGOギフトを選択することになります。開始デッキの主要なダメージタイプに一致するものを選びましょう。
  </Step>
  <Step num="4" title="ルートを計画する">
    「エリート」や「チャレンジ」ノードを探しましょう。これらは標準的なエンカウントに比べて、高ティアのEGOギフトをドロップする確率が高くなっています。
  </Step>
  <Step num="5" title="シナジーとデフラグメント">
    攻略の合間に、収集した素材を使ってお気に入りの衣装をデフラグメントし、特定のギフトをより効果的にするパッシブを解放します。
  </Step>
</Steps>

<Callout type="success" title="戦略の成功">
熟練したプレイヤーは、凡庸なギフトを無視して、より多くのカード操作を可能にする「ユーティリティ」の選択肢を優先することがよくあります。デッキ構築ゲームにおいて、一貫性は純粋なパワーよりも重要な場合が多いのです。
</Callout>

</Section>
