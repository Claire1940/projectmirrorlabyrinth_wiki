<Section icon={Sparkles} title="Project Mirror Labyrinth 破裂ビルドの基本">

**Project Mirror Labyrinthの破裂（Rupture）ビルド**は、Robloxのローグライト・デッキビルドゲームである本作において、現在最も高難易度ダンジョンの攻略に効果的な戦略の一つです。『Limbus Company（リンバス・カンパニー）』や『Library of Ruina（ライブラリー・オブ・ルイナ）』の戦闘システムから強い影響を受けており、敵に「破裂」ステータス効果を付与することを中心に展開します。高い基礎攻撃力に依存する純粋なダメージビルドとは異なり、破裂特化のデッキは、多くの防御層を無視して一貫した追加ダメージを与えることで真価を発揮します。破裂をスタックさせることで、その後に与える攻撃の1ヒットごとに破裂のスタック数に等しいボーナスダメージが加算されるため、多段ヒットカードの威力が飛躍的に向上します。

**動画のハイライト:**
- **ビルドの核心**: 破裂スタックとヒット数の相互作用を理解する。
- **カードの選択**: どの攻撃カードが最も効率よく破裂を付与できるか。
- **衣装のシナジー**: 高ティアのダンジョン攻略において、特定の「鏡映（Reflections）」が必須となる理由。

このビルドをマスターするには、破裂の付与（スタックの追加）と破裂の強度（スタックが早く減衰しないようにすること）のバランスを理解する必要があります。現在の2026年のメタでは、プレイヤーはデバフを常に維持し続けられるような、攻撃的なデッキサイクルに焦点を当てています。

| メカニクス | 破裂ビルドの焦点 | メリット |
| :--- | :--- | :--- |
| **ダメージタイプ** | 固定追加ダメージ | 敵のダメージ軽減を無視 |
| **ヒット数** | 高多段ヒットカード | 破裂ダメージを複数回誘発 |
| **スケーリング** | スタック依存 | 戦闘が長引くほどダメージが増加 |
| **持続性** | 防御カードのバランス | 倒されることなくスタックを蓄積可能 |
| **リソース消費** | アップグレード用の欠片 | 受動的な破裂付与効果を強化 |

<Callout type="tip" title="攻略のヒント">
ラン（挑戦）を開始する際は、「火花（Spark）」や「裂傷（Lacerate）」系のカードを探しましょう。これらは、強力なアタッカーカードを繰り出す前に、初期の破裂スタックを付与するための土台として非常に役立ちます。
</Callout>

<Section icon={Trophy} title="必須の衣装と鏡映">

Project Mirror Labyrinthにおいて、衣装（Outfit）の選択はパッシブ能力と基礎ステータスを決定づけます。**Project Mirror Labyrinthの破裂ビルド**では、適当な衣装を選ぶわけにはいきません。ステータス効果の付与を特に強化するものや、速度ボーナスを提供するものが必要です。敵が行動する前に破裂を付与することで、こちらの防御的なカウンターでも追加ダメージを誘発できるようになるため、速度は極めて重要です。「特定抽出（Targeted Reflections）」メニューで欠片（Fragments）を使用することが、これらの特定の衣装を狙う上で最も効率的な方法です。

  <Card title="速度特化の衣装">

    - **回避ボーナス**: 重いブロックカードの必要性を減らします。
    - **シナジー**: 「クイックドロー」タイプのカードと最も相性が良いです。
  </Card>
  <Card title="多段ヒットスペシャリスト">

    - **シナジー**: 高度なエネルギー管理が必要です。

  <Card title="デバフ増幅器">

  </Card>
</CardGrid>

衣装の「解析（Defragmenting）」プロセスこそが、真の力を引き出す鍵です。「凝縮（Condense）」や「光の種（Seeds of Light）」などの素材を使用することで、破裂スタックを永続化させたり、範囲（AOE）ダメージとして爆発させたりするパッシブを解放できます。

| 衣装ティア | 推奨用途 | 重要ステータス | 解析優先度 |
| :--- | :--- | :--- | :--- |
| **Sティア (雷衡)** | エンドゲームダンジョン | 速度 / ヒット数 | **最大** |
| **Aティア (標準)** | 中盤の素材集め | 破裂強度 | **高** |
| **Bティア (初期)** | 序盤の進行 | 基本防御 | **低** |

<Callout type="warning" title="リソース管理">
Bティアの衣装に「光の種」を無駄遣いしないでください。破裂ビルドが2026年のエンドゲームコンテンツに対応できるように、SR以上の「鏡映」のために温存しておきましょう。
</Callout>

<Section icon={Swords} title="コアデッキ構築とカードの組み合わせ">

**Project Mirror Labyrinthの破裂ビルド**を成功させるには、細部まで調整されたデッキが必要です。多段ヒットの攻撃カードと、価値の高い防御カードの比率を2:1にすることを目指します。破裂ダメージはヒットごとに誘発されるため、敵に10以上の破裂スタックがたまっている場合、1回で15ダメージを与えるカードよりも、5ダメージで3回ヒットするカードの方が圧倒的に強力になります。

| カード名 (タイプ) | 効果 | 採用理由 |
| :--- | :--- | :--- |
| **ラピッドスラスト (攻撃)** | 3ヒット、低い基礎ダメージ | 1回の使用で破裂を3回誘発 |
| **鏡の破片 (攻撃)** | 破裂を4付与 | ボス戦における主要なスタック生成源 |
| **ガラスの盾 (防御)** | ブロック + カウンターヒット | 敵のターン中に破裂を誘発 |
| **粉砕 (スキル)** | 破裂を5消費して2倍ダメージ | 低HPの敵に対するトドメ役 |
| **反響する一撃 (攻撃)** | 直前の攻撃カードを繰り返す | 破裂の付与回数を2倍に |

Project Mirror Labyrinthにおける「攻撃」カードと「防御」カードの相互作用は独特です。「カウンター」のキーワードを持つ防御カードを使用した場合、敵のターン中に繰り出される反撃ヒットでも破裂ダメージが誘発されます。これにより、防御的に立ち回りながらもダメージを与えることが可能となり、これがプロレベルのプレイヤーの証となります。

<Callout type="success" title="プロコンボ">
**鏡の破片**でスタックを付与し、続いて**反響する一撃**でスタックを倍増させます。ターンの最後は**ガラスの盾**で締めくくり、自身の安全を確保しつつ、敵がこちらのブロックに対して自滅するのを待ちましょう。
</Callout>

</Section>

<Section icon={Target} title="段階的な進行ガイド">

究極の**Project Mirror Labyrinth 破裂ビルド**は、一回の挑戦（ラン）で完成するものではありません。リソース収集とカードドラフトへの計画的なアプローチが必要です。現在の2026年バージョンのゲームでキャラクターを最適化するために、以下のステップに従ってください。

<Steps>
  <Step num="1" title="初期リソースの収集">
    まずは有効なコードをすべて引き換えることから始めましょう。**OfficeDelay** や **UrbanPlague** などのコードから得られる「欠片（Fragments）」を使用して、最初の「鏡映（Reflections）」を行います。速度またはヒット数ボーナスを提供する衣装を少なくとも1つ入手することを目指します。
  </Step>
  <Step num="2" title="ダンジョン入場と序盤のドラフト">
    「立ち入り禁止（NO ENTRY）」ゲートまたはTABメニューからダンジョンに入場します。最初の数フロアでは、「多段ヒット」または「破裂」のキーワードを持つカードのドラフトを最優先してください。スタックメカニクスに貢献しない、一般的な高ダメージカードは避けましょう。
  </Step>
  <Step num="3" title="ラン中盤の最適化">
    ステータス効果を強化する「E.G.Oギフト」を探します。これらのパッシブアイテムは、破裂スタックの効果を倍増させることがよくあります。ショップに遭遇した場合は、デッキの一貫性を高めるために、基本的な「ストライク」カードの削除を優先してください。
  </Step>
  <Step num="4" title="衣装の解析">
    ランが終了したら、集めた「凝縮（Condense）」と「光の種（Seeds of Light）」を使って、メインの衣装を「解析（Defragment）」します。最初の2つのパッシブスロットの解放に集中しましょう。これらは通常、破裂ビルドにおいて最大のパワー急上昇をもたらします。
  </Step>
</Steps>

| 進行のマイルストーン | 報酬 / 目標 | 優先度 |
| :--- | :--- | :--- |
| **レベル10** | 特定抽出の解放 | **極めて重要** |
| **最初のボス撃破** | メガ鏡映チケットの獲得 | **高** |
| **採光の解放** | 凝縮（Condense）の安定した周回 | **高** |
| **衣装の完全解析** | 破裂パッシブの最大化 | **エンドゲーム** |

<Callout type="info" title="メンテナンスの注意点">
Project Mirror Labyrinthは毎週火曜日の午後2:00頃にメンテナンスが行われます。進行状況が失われるのを避けるため、長時間のダンジョン挑戦は計画的に行いましょう。
</Callout>
