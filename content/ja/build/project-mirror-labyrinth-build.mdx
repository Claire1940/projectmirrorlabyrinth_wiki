### ビルドの核心的な基礎
このゲームにおけるあらゆるビルドの基礎は、「デッキ」、「衣装」、そしてデフラグによって解放される「パッシブスキル」の3つの柱に基づいています。プレイヤーはすべてのダンジョン攻略を、現在の構成のテストとして扱う必要があります。装備が固定されている従来のRPGとは異なり、デッキは攻略中に進化するため、開始時の衣装はランダム性の高いカードドラフトに対応できる汎用性の高いベースを提供しなければなりません。

</Section>

<Section icon={Swords} title="戦闘メカニズムとカードのアーキタイプ">

Project Mirror Labyrinthの戦闘は、攻撃カードと防御カードの戦略的な使用を中心に展開します。毎ターン、受けるダメージを軽減するか、キルを狙いに行くかを決定する選択肢が提示されます。バランスの取れた**Project Mirror Labyrinth ビルド**は、通常、攻撃機能と防御機能の比率を60/40に維持します。

| カードカテゴリ | 主な機能 | 理想的な使用場面 | 戦略的価値 |
| :--- | :--- | :--- | :--- |
| **攻撃カード** | 直接ダメージを与える | 敵を素早く仕留める | 雑魚敵の掃討に**高い** |
| **防御カード** | ダメージをブロック/軽減 | ボスの強力な一撃を耐える | 長期戦において**極めて重要** |
| **ユーティリティ** | ドロー/ステータス強化 | 強力なコンボの準備 | デッキ回転に**中程度** |
| **EGOカード** | 特殊な高影響力アクション | 戦況を逆転させる | 使用制限があるが**非常に高い** |

<Callout type="warning" title="戦闘の警告">
防御カードを無視することは、攻略を終わらせる最短の方法です。ラビリンスの深層にいるボスは、不完全なビルドを貫通する多段ヒット攻撃を繰り出してきます。ビルドには少なくとも3枚の信頼できる「ブロック」または「回避」カードが含まれていることを確認してください。
</Callout>

### カードと衣装のシナジー
衣装は単なる見た目のスキンではありません。これらはゲームにおける「クラス」システムです。各衣装には、特定のカードタイプを強化する固有のステータス補正が備わっています。例えば、「斬撃」耐性が高く「貫通」ダメージボーナスを持つ衣装は、貫通攻撃に焦点を当てたデッキと組み合わせるべきです。これらのタグを理解することは、キャラクターの出力を最適化するために不可欠です。

| 衣装タイプ | 相性の良いカード | ビルドの焦点 |
| :--- | :--- | :--- |
| **ストライカー** | 高火力の物理カード | 瞬間火力 / 高速掃討 |
| **ガーディアン** | シールドおよびパリィカード | タンク / 消耗戦 |
| **タクティシャン** | カードドローとエネルギー回復 | コンボ重視 / 無限ターン |
| **EGO特化** | 特殊なEGOアビリティカード | 高リソース / 終盤のパワー |

</Section>

### デフラグ（Defragmenting）プロセス
強力な衣装を手に入れたら、次のステップはデフラグです。このプロセスには、**凝縮体（Condense）**や**光の種（Seeds of Light）**といった希少な素材が必要です。デフラグを行うことで、その衣装に永続的なパッシブアビリティが解放され、より難易度の高いダンジョンで大きな優位性を得ることができます。

| 素材 | 主な入手先 | ビルドでの用途 |
| :--- | :--- | :--- |
| **欠片** | ダンジョン攻略 & コード | 新しい鏡像の抽出 / スキル |
| **凝縮体** | 高レベル報酬 | 初期のデフラグティア |
| **光の種** | 特殊イベント & エリートドロップ | 最終パッシブティアの解放 |
| **チケット** | マイルストーン報酬 | 衣装の確定入手 |

</Section>

<Section icon={Target} title="初めてのダンジョン攻略の始め方">

ラビリンスでの旅を始めるのは、最初は戸惑うかもしれません。このゲームはProject Moonシリーズの容赦ない性質を反映しており、手取り足取り教えてくれることはありません。以下の手順に従って、最初の数回の攻略を実りあるものにし、適切なビルドに必要なリソースを集めましょう。

<Steps>
  <Step num="1" title="初期準備">
    Robloxでゲームを起動します。迷路に入る前に、画面左側の**コードメニュー**を開きます。**Welcome**などのスターターコードを入力して、最初の欠片と光の種のセットを受け取りましょう。
  </Step>
  <Step num="2" title="衣装の装備">
    キャラクターメニューに移動し、現在持っている中で最高の衣装を装備します。デフォルトの初期衣装であっても、その基本的な強みを理解していれば効果的です。ステータスボーナスを確認し、攻撃と防御のどちらに集中すべきか判断してください。
  </Step>
  <Step num="3" title="ラビリンスへの入場">
    選択肢は2つあります。**TABキー**を押して「Play」を選択するか、ロビーにある**NO ENTRYゲート**まで歩いていきます。どちらの方法でも、ダンジョンのマッチメイキングまたはソロ入場プロセスが開始されます。
  </Step>
  <Step num="4" title="デッキのドラフト">
    最初の数部屋を進むと、カードの選択肢が提示されます。装備している衣装のタグに一致するカードを選んでください。貫通特化の衣装を着ているなら、「貫通」キーワードを持つカードを優先しましょう。
  </Step>
  <Step num="5" title="リソース管理">
    戦闘の合間に、休憩所やショップを見つけることがあります。これらを利用して、デッキから弱いカードを「パージ（削除）」したり、頻繁に使用するカードをアップグレードしたりしましょう。肥大化したデッキよりも、スリムで効率的なデッキの方が常に強力です。
  </Step>
</Steps>

<Callout type="success" title="新規プレイヤーの成功">
最初の「採光（Luxcavation）」または基本的なダンジョン攻略を完了すると、少なくとも1回は通常鏡像を引くのに十分な欠片が得られます。すぐにこれを使用して、初期装備に代わる高レアリティの衣装を狙いましょう。
</Callout>

</Section>

<Section icon={Shield} title="ラビリンス生存のための高度な戦略">

ゲーム中盤から終盤に進むにつれて、**Project Mirror Labyrinth ビルド**は複雑な敵のメカニズムに対応する必要があります。敵は「出血」や「火傷」といった状態異常を使用し始め、最初の一撃をブロックしても体力を削られるようになります。

### 状態異常の管理
状態異常への対策を理解することは不可欠です。一部の衣装は自然な耐性を提供し、特定のカードはこれらのデバフを「浄化」できます。

| 状態異常 | プレイヤーへの影響 | 最適な対策 |
| :--- | :--- | :--- |
| **出血** | カードを使用するたびにダメージ | 価値の高いカードを使い、総プレイ数を減らす |
| **火傷** | ターン終了時にダメージ | 戦闘を素早く終わらせる / 浄化カードを使用 |
| **麻痺** | カードの効率が低下 | 状態が解けるまで防御カードを優先する |
| **脆弱** | 受けるダメージが増加 | 防御カードやシールドを多めに重ねる |

<Callout type="tip" title="プロ戦略">
ダンジョンの後半ステージでは、「カードドロー」系カードを優先してください。1ターンに見ることができるカードの枚数が多いほど、ボスの究極攻撃を生き延びるために必要な特定の防御ツールを見つけられる可能性が高まります。
</Callout>

### 毎週のメンテナンスとアップデート
開発者は通常、**毎週火曜日の午後2時（P.M.）頃**にメンテナンスを実施します。この間、サーバーが不安定になったり、アクセスできなくなったりすることがあります。現在のメタビルドに影響を与える可能性のある最新の追加コンテンツを確認するために、常にRobloxのゲームタイトル（現在は **[LEI HENG + NOT BINAH]** などのタグが表示されています）をチェックしてください。

</Section>
