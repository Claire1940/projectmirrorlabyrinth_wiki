<Section icon={Swords} title="服装与镜像：构建的基础">

服装是你 **Project Mirror Labyrinth 最强构建**中最关键的组成部分。与你在冒险中找到的临时卡牌不同，服装提供永久的属性加成和独特的被动能力，这些定义了你角色的基础力量。它们通过“镜像系统”获得，这是一种你需要消耗碎片或票据的扭蛋式机制。

| 服装类别 | 获取成本 | 最佳适用场景 |
| :--- | :--- | :--- |
| **定向镜像** | 高额碎片成本 | 追求特定的 Meta 定义服装 |
| **标准镜像** | 中等碎片成本 | 账户基础成长与卡牌多样性 |
| **SR 保底** | 需要特殊票据 | 保证获得高稀有度力量提升 |
| **碎片化 (Defragmented)** | 浓缩物 + 光之种 | 现有服装的后期优化 |

在构建角色时，你应该寻找与特定卡牌类型协同的服装。例如，提供“格挡”数值加成的服装对于防御型构建至关重要，而触发“穿刺”卡牌后进行追击的服装则是进攻型构建的理想选择。碎片化过程通过解锁可以彻底改变地下城后期构建方式的隐藏被动，进一步增强这些特性。

<Callout type="warning" title="维护公告">
开发人员大约在**每周二下午 2:00** 进行维护。在此期间，镜像系统和地下城入口可能无法使用。请相应地规划你的碎片支出，以免在服务器重置期间丢失进度。
</Callout>

</Section>

<Section icon={Shield} title="战斗策略与卡牌选择">

Project Mirror Labyrinth 中的战斗是一场战术博弈。你不仅仅是在出牌；你是在对一个活的迷宫做出反应。“最强构建”的好坏取决于玩家解读局面的能力。你必须平衡你的牌组，以确保始终有应对敌人下一步行动的手段。

<Steps>
  <Step num="1" title="分析敌人意图">
    查看敌人上方的图标。他们是在攻击还是防御？利用这些信息来决定是打出护盾卡还是攻击卡。
  </Step>
  <Step num="2" title="管理牌组大小">
    在 Roguelite 游戏中，一套小而精、更稳定的牌组通常比一套充满平庸卡牌的大牌组更好。尽可能移除弱小的初始卡牌。
  </Step>
  <Step num="3" title="与被动协同">
    始终打出能触发服装被动能力的卡牌。如果你的服装奖励打出三张同色卡牌，请优先考虑该序列。
  </Step>
</Steps>

| 卡牌类型 | 在构建中的角色 | 战略价值 |
| :--- | :--- | :--- |
| **攻击卡** | 伤害输出 | 快速清理战斗的必需品 |
| **防御卡** | 伤害减轻 | 长时间地下城冒险中的生存关键 |
| **功能卡** | 资源/牌组操控 | 帮助循环到最强大的卡牌 |
| **EGO 卡** | 高冲击力终结技 | 能够扭转败局 |

了解卡牌库至关重要。有些卡牌起初看起来很弱，但与合适的 EGO 礼物或服装被动搭配时会成为强力武器。随着你在迷宫中深入，你将有机会向牌组中添加新卡牌。始终问自己：“这张卡能改善我当前的构建，还是会稀释我的策略？”

<Callout type="info" title="专业提示">
如果你发现自己被困在某个特定楼层，请尝试重新评估你的防御卡牌。许多玩家过于关注伤害，结果被中期 Boss 的高伤害输出压垮。
</Callout>

</Section>
