<Section icon={Swords} title="戦闘メカニクス：カードタイプとシナジー">

Project Mirror Labyrinthの戦闘はターン制のカードバトルです。デッキはダンジョン攻略中の選択によって成長も衰退もする「生きた存在」です。都市荒廃（Urban Plague）以降の後半ステージを生き抜くためには、各カードの役割の違いを理解することが極めて重要です。

### 必須カードカテゴリー

| カードタイプ | 主な機能 | 戦略的な用途 |
| :--- | :--- | :--- |
| **攻撃（Attack）** | 敵に直接ダメージを与える。 | 敵の防御が低い時や、脅威を素早く仕留める時に使用。 |
| **防御（Defense）** | 受けるダメージを軽減する/防ぐ（ブロック）。 | ボスの「大技」ターンを耐えるために必須。常に手札に2枚はキープすること。 |
| **ユーティリティ（Utility）** | バフ、カードドロー、またはリソース獲得。 | デッキを回転させ、「勝利条件（Win Condition）」となるカードを手札に呼び込むために不可欠。 |
| **EGO** | 高コスト・高威力のフィニッシャー。 | 真の力を引き出すには、EGOチケットまたは特定の進行度が必要です。 |

<Callout type="success" title="デッキ構築のヒント">
バランスの取れたデッキは、通常、攻撃カードと防御カードの比率を**3:2**に維持します。攻撃に偏りすぎると、迷宮の3層目で発生する高ダメージの急襲（スパイク）に対して無防備になってしまいます。
</Callout>

</Section>

<Section icon={Target} title="最初のダンジョンランの始め方">

Project Mirror Labyrinthでの旅立ちは、その独特な雰囲気を持つミニマルなUIのせいで戸惑うかもしれません。ロビーで迷子にならないよう、以下の手順に従ってください。

<Steps>
  <Step num="1" title="ロビーでの準備">
    迷宮に入る前に、必ず[公式Robloxコミュニティ](https://www.roblox.com/communities/127473071/Project-Mirror-Labyrinth)に参加してください。これは、初期フラグメントを入手するためのコードを引き換える際の必須条件となっています。
  </Step>
  <Step num="2" title="初期コードの引き換え">
    画面左側のメニューを開きます。コード「**Welcome**」を入力すると、350フラグメント、500凝縮物（Condense）、3つの光の種（Seeds of Light）を受け取ることができます。これにより、最初の数回の鏡映を行うのに十分なリソースが手に入ります。
  </Step>
  <Step num="3" title="ダンジョンへのアクセス">
    2つの方法があります。キーボードの**TAB**キーを押して「Play」を選択するか、ロビーにある物理的な「**NO ENTRY（進入禁止）**」ゲートまで歩いて行ってインタラクトします。どちらの方法でも、迷宮の開始階層へとテレポートします。
  </Step>
  <Step num="4" title="初期デッキの選択">
    自分の好みのプレイスタイルに合った初期衣装を選びましょう。安全性を重視する場合は、高い**ブロック（Block）**補正を持つ衣装を探してください。スピード重視の場合は、**斬撃（Slash）や貫通（Pierce）**のダメージボーナスを優先しましょう。
  </Step>
</Steps>

<Callout type="info" title="メンテナンススケジュール">
ダンジョンに入れない場合は、時間を確認してください。ゲームは毎週**火曜日の午後2:00（UTC）**にメンテナンスを実施します。この時間帯は通常1〜2時間サーバーがダウンします。
</Callout>

</Section>
