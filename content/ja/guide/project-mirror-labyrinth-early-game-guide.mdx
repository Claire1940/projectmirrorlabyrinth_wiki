<Section icon={Swords} title="戦闘の基礎：精神力、マッチ、混乱耐性">

Project Mirror Labyrinthの戦闘はカードベースの「マッチ（Clash）」を中心に展開します。ダメージを与え合うだけの伝統的なRPGとは異なり、このゲームのすべての攻撃は敵によって対抗される可能性があります。これらのマッチに勝つことが、序盤を生き残るための最も重要な要素です。

### 混乱耐性とガードの力
キャラクターは混乱耐性（Posture）を消費して行動を実行します。混乱耐性がゼロになると「混乱（Stagger）」状態になり、非常に脆弱になります。手札の選択肢が限られている序盤では、このリソースの管理が不可欠です。

<CardGrid cols={3}>
  <Card title="マッチ勝利">
    - **精神力の生成**: マッチに勝つことが精神力を得る主な手段です。
    - **ダメージ防止**: 敵のカード効果の発動を防ぎます。
    - **モメンタム**: 攻勢のプレッシャーを高く維持します。
  </Card>
  <Card title="防御ガード">
    - **混乱耐性コスト0**: 標準的なガードはコストなしで使用できます。
    - **混乱耐性の回復**: 使用時に混乱耐性を3ポイント回復します（特定のパークで5ポイント）。
    - **安全なターン**: 手札が悪い時の防御的なターンに最適です。
  </Card>
  <Card title="呼吸とクリティカル">
    - **ダメージ倍率**: 呼吸（Poise）は総出力を最大1.5倍まで高めます。
    - **斬撃シナジー**: 斬撃中心の衣装と非常に相性が良いです。
    - **注意**: 序盤から呼吸に投資しすぎないこと。まずは純粋なマッチ威力を優先しましょう。
  </Card>
</CardGrid>

各戦闘アクションがリソースにどのように影響するかについては、以下の比較表を参照してください：

| 戦闘アクション | 混乱耐性コスト | 精神力への影響 | 主な用途 |
| :--- | :--- | :--- | :--- |
| **通常攻撃** | 変動（通常1-3） | マッチ勝利時に上昇 | ダメージを与え、マッチに勝つため |
| **ガードカード** | 0 | 変化なし | 混乱耐性の回復と攻撃の吸収 |
| **E.G.Oスキル** | 高（12-15） | 高コスト / 高リターン | 出目の高いマッチに勝つため |
| **廃棄アクション** | 変動 | 変化なし | 手札の入れ替えと廃棄パッシブの発動 |

<Callout type="warning" title="混乱警告">
混乱耐性を完全に枯渇させないでください。敵が勝てそうにない高い出目の攻撃を準備している場合は、弱い攻撃で対抗するのではなく、ガードカードを使用してダメージを抑えましょう。
</Callout>

</Section>

<Section icon={Target} title="ステップバイステップ序盤進行ガイド">

効率的に進行するためには、構造化されたルートを辿るべきです。裏路地に必要以上に長居しないでください。本当のパワーアップは迷宮のより深くにあります。

<Steps>
  <Step num="1" title="初期資金の受け取り">
    設定メニューで **Welcome** や **OfficeDelay** などの有効なコードをすべて入力し、数千の破片、凝縮物、光の種を確保します。
  </Step>
  <Step num="2" title="強力な初期衣装の入手">
    破片を通常抽出（Standard Reflection）またはターゲット抽出（Targeted Reflection）に使用し、信頼できる衣装を引き当てます。序盤のマッチを楽にするため、カードの基本値が高い衣装を探しましょう。
  </Step>
  <Step num="3" title="裏路地のクリア">
    TABメニューまたは「NO ENTRY」ゲートから裏路地ダンジョンに入ります。敵のパターンを学習し、ガードカードを使って混乱耐性を管理することに集中しましょう。
  </Step>
  <Step num="4" title="HP回復パッシブを狙う">
    裏路地で「ふらつくフィクサー（Swaying Fixer）」や「マントのならず者（Cape Goon）」などの特定の敵を狙います。彼らはHPを再生するパッシブをドロップします。
  </Step>
  <Step num="5" title="図書館への進出">
    裏路地のボスを倒したら、すぐに図書館へ向かいましょう。このゾーンには、ゲーム内で最高の永続パッシブをドロップする敵が登場します。
  </Step>
</Steps>

<Callout type="info" title="進行のヒント">
ボス戦前の報酬選択では、リソースが少ない場合はHP再生や混乱耐性回復を優先してください。デッキが基本カードで溢れかえっている場合のみ、カードの削除や廃棄の選択肢を選びましょう。
</Callout>

</Section>

<Section icon={Sparkles} title="序盤のおすすめパッシブと衣装">

パッシブは裏路地や図書館で特定の敵を倒すことで解放されます。衣装に適切なパッシブを装備することで、デッキのパフォーマンスが劇的に変わります。

| パッシブ名 | 出現エネミー | 主な効果 | 推奨ビルドタイプ |
| :--- | :--- | :--- | :--- |
| **孤高のフィクサー** | 裏路地の事務所 | 単独戦闘時に攻撃力を強化 | ソロ / 初心者ビルド |
| **弱さを通して歌う** | D7協会 | 精神力が低い時にマッチ威力を増加 | ハイリスクなマッチデッキ |
| **マージン** | 図書館の廃棄系エネミー | 手札が空の時にカードを引く | 廃棄多用の衣装 |
| **指令** | 図書館の人指し指系エネミー | 戦闘中の速度ダイス（アクションスロット）を+1する | 全ビルド（強く推奨） |

### 衣装選択の戦略
序盤に使用する衣装を選ぶ際は、火傷や出血などの複雑な状態異常よりも、純粋な「マッチの強さ（出目の高さ）」を重視しましょう。基本パワーの高いカードは、運が悪くても序盤のゾーンを突破させてくれます。

| 衣装ティア | 衣装名 | 主な強み | アップグレード優先度 |
| :--- | :--- | :--- | :--- |
| **Sティア** | KOプラスツー警備主任 | 高い出目、優れた火傷付与、強力なガード | 高（すぐにデフラグ推奨） |
| **Aティア** | 近代戦争第7事務所 | 安定した斬撃ダメージと堅実なマッチ威力 | 中 |
| **Bティア** | 第7級事務所 | まともな初期デッキだが、強力なパッシブに欠ける | 低（可能なら入れ替え） |

<Callout type="success" title="アップグレード推奨">
光の種と凝縮物を使用して、Sティアの衣装を**デフラグ（Defragment）**しましょう。これにより、ダンジョン攻略中ずっと有効な潜在パッシブ能力が解放されます。
</Callout>

</Section>
