<Section icon={Map} title="Project Mirror Labyrinth を始める">

Project Mirror Labyrinth は、戦術的なカードバトルと雰囲気のあるローグライト探索を融合させた、Roblox 独自の体験です。従来の RPG とは異なり、攻略の成功はラン中のデッキ構築と、探索の合間のリソース管理に大きく依存します。迷宮に飛び込む前に、基本的な経済システムと、最初の報酬を確保するために必要な手順を理解しておく必要があります。

**動画のハイライト:**
- Library of Ruina と Limbus Company を融合させたゲームプレイの概要。
- ダンジョン入り口のゲートとロビー UI のビジュアルガイド。
- 実際のカードベースの戦闘システムのデモンストレーション。

<YouTubeEmbed videoId="UvVyOdtVVvM" title="Roblox Project Mirror Labyrinth Gameplay" />

旅を始めるには、まずスターターリソースを集めることに集中すべきです。ゲーム内には、主要な進行メカニクスである「リフレクション」システムの解放に不可欠な複数の通貨が存在します。

| 資源 | 主な用途 | 入手方法 |
| :--- | :--- | :--- |
| **欠片 (Fragments)** | リフレクションでの衣装の入手 | コード、ダンジョン攻略、ミッション |
| **凝縮 (Condense)** | パッシブ解放のための衣装のデフラグ | 特別イベント、スターターコード |
| **光の種 (Seeds of Light)** | 高レベルの衣装の成長 | マイルストーン報酬、コード |
| **リフレクションチケット** | 常設/ピックアップバナーでの無料ガチャ | メンテナンス補填、イベント |

<Callout type="tip" title="プロのヒント">
コードを引き換える前に、必ず Project Mirror Labyrinth の公式 Roblox グループに参加し、ゲームに「いいね」をしてください。多くの報酬は、これらのコミュニティ要件を満たさないとロックされたままになります。
</Callout>

<Section icon={Swords} title="戦闘メカニクスとデッキ構築戦略">

Project Mirror Labyrinth における戦闘はターン制であり、手札からカードを出して敵の行動に対処することを中心に展開します。ダンジョンを進むにつれてデッキに新しいカードが追加され、その相乗効果（シナジー）が攻略の成否を分けることになります。カードタイプの違いを理解することが、メタをマスターするための第一歩です。

<CardGrid cols={3}>
  <Card title="攻撃ビルド">
    - **高ダメージ**: 敵を素早く圧倒することに特化。
    - **脆弱**: 致命的なカウンターを避けるために正確なタイミングが必要。
    - **最適**: 高速周回やダンジョンの低層階。
  </Card>
  <Card title="防御ビルド">
    - **持続性**: ブロックや回避カードを使用してダメージを軽減。
    - **長期戦**: 戦闘時間は長くなるが、非常に安全。
    - **最適**: ボス戦や高難易度の迷宮。
  </Card>
  <Card title="ユーティリティ・ハイブリッド">
    - **適応力**: リソース生成と適度なダメージを混合。
    - **複雑**: 操作は難しいが、多様な敵タイプに対応可能。
    - **最適**: 深層部を目指す経験豊富なプレイヤー。
  </Card>
</CardGrid>

カードシステムは *Slay the Spire* にインスパイアされており、「エネルギー」または「コスト」を効果的に管理する必要があります。高コストの攻撃カードを使いすぎると、敵のターン中に無防備になってしまう可能性があります。

| カードカテゴリー | 効果タイプ | 戦略的価値 |
| :--- | :--- | :--- |
| **攻撃 (Attack)** | 直接ダメージ | **高**: 戦闘を素早く終わらせるために必要。 |
| **防御 (Defense)** | シールド / ブロック | **重要**: 敵のターン中の HP 減少を防ぐ。 |
| **バフ (Buff)** | ステータス上昇 | **中**: 長期的な戦闘効率を高める。 |
| **EGO** | 究極奥義 | **最高**: 劣勢を一気に覆すことができる。 |

<Callout type="warning" title="戦闘の警告">
防御カードを無視しないでください。Project Mirror Labyrinth では、フロアごとに体力が全回復するわけではありません。「ガラスの砲台」ビルドは、最終ボスに到達する前に力尽きることがよくあります。
</Callout>

</Section>

<Section icon={Shield} title="ダンジョン探索とメンテナンス">

**Project Mirror Labyrinth 攻略**の核心は、迷宮のようなダンジョンをナビゲートすることです。これらはプロシージャル（手続き型）に生成されるため、進む道はランごとに変化します。戦闘、エリートとの遭遇、または休憩所など、異なるノードを選択する必要があります。

<Steps>
  <Step num="1" title="ランの開始">
    **TAB** キーを押してゲーム内メニューを開き「Play」を選択するか、ロビーにある **NO ENTRY** ゲートまで歩いて行って開始します。
  </Step>
  <Step num="2" title="ルートの選択">
    マップを確認して経路を計画します。序盤はデッキを安定させるために、カード報酬が得られるノードを優先しましょう。
  </Step>
  <Step num="3" title="遭遇戦の管理">
    カードバトルを行います。勝利して欠片や新しいカードの選択肢を獲得しましょう。
  </Step>
  <Step num="4" title="採光（Luxcavation）へ到達">
    より深い層へと進み、上位ティアの報酬やボス戦を解放しましょう。
  </Step>
</Steps>

ゲームは安定性の確保や「Lei Heng」または「Not Binah」アップデートのような新コンテンツ追加のために、定期的なメンテナンスが行われることに注意してください。これらの期間中は、ダンジョンへのアクセスが制限される場合があります。

| イベント | 標準時間 (EST) | プレイヤーへの影響 |
| :--- | :--- | :--- |
| **週次メンテナンス** | 火曜日 午後 2:00 | サーバーオフライン、進行状況は保存されません。 |
| **パッチ適用** | 不定期 / 週末 | 新しいコードのリリース、メタの変更。 |
| **デイリーリセット** | 午前 0:00 | ミッションの更新、ログイン報酬。 |

<Callout type="info" title="システム情報">
メンテナンス時間外に「NO ENTRY」エラーが発生した場合は、新しいサーバーに入り直してみてください。Roblox のアップデートにより、ロビーのインスタンスがゲームのバックエンドと同期しなくなることがあります。
</Callout>

</Section>

<Section icon={Sparkles} title="リフレクションと衣装の進行">

リフレクション（Reflections）は、欠片を消費して**衣装（Outfits）**を獲得するガチャ形式のシステムです。衣装は単なる見た目の変更ではなく、プレイスタイルを決定づける大幅なステータスブーストとユニークなパッシブ能力を提供します。

リフレクションバナーには主に **常設（Standard）** と **ピックアップ（Targeted）** の 2 種類があります。ピックアップバナーは特定の高ティア衣装の排出率が高いため、特定のビルドを最適化したいプレイヤーに適しています。

| 通貨 / アイテム | 入手方法 | 最適な使用方法 |
| :--- | :--- | :--- |
| **メガチケット** | レアコード / イベント | リフレクションでの SR 以上確定ガチャ。 |
| **EGO チケット** | 高難易度ダンジョン | 強力な EGO 能力の解放。 |
| **欠片 (Fragments)** | 通常のプレイ | 標準的なリフレクションガチャの購入。 |
| **光の種 (Seeds of Light)** | 実績マイルストーン | 衣装の最終段階のデフラグ。 |

<Callout type="success" title="進行のコツ">
複数の低レベルの衣装を持つよりも、一つの衣装に集中して完全にデフラグする方が多くの場合有利です。デフラグによって解放されるパッシブ能力は、ゲームバランスを大きく変える力を持っています。
</Callout>

</Section>
