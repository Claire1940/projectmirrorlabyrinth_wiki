<Section icon={Target} title="ダンジョン入場と初心者向けステップ">

Project Mirror Labyrinthでの最初の挑戦（ラン）を始めるのは簡単ですが、ゲームロビーでの手引きは最小限です。ダンジョン挑戦を開始し、デッキ構築を始めるには2つの異なる方法があります。

<Steps>
  <Step num="1" title="Robloxグループへの参加＆ゲームへの「いいね」">
    ゲームを起動する前に、Project Mirror Labyrinthの公式Robloxページにアクセスしてください。エクスペリエンスに「いいね」を押し、公式開発者グループに参加します。これは必須の手順です。そうしないと、ロビーでコードを引き換えようとしてもゲーム側で拒否されます。
  </Step>
  <Step num="2" title="ゲームの起動とウェルカムコードの引き換え">
    エクスペリエンスを開き、**TABキー**を押すか、左側にあるメニューアイコンをクリックして「Codes（コード）」メニューにアクセスします。スターターコード「**Welcome**」を入力すると、すぐに350 Fragments、500 Condense、3 Seeds of Lightを受け取ることができます。
  </Step>
  <Step num="3" title="ダンジョンゲートへのアクセス">
    挑戦を始めるには、**TABキー**を押して「**Play**」を選択するか、ロビーにある光る物理的な**NO ENTRY（進入禁止）ゲート**に近づいてインタラクトし、マッチングキューに入ります。
  </Step>
  <Step num="4" title="スターターデッキのドラフト">
    迷宮に入ると、攻撃カードと防御カードの基本的なプールを受け取ります。最初の目標は、受けるダメージを抑えつつ、着実に攻撃を与えるために、デッキのバランスを整えることです。
  </Step>
</Steps>

<Section icon={Swords} title="戦闘メカニクスとカードシナジー">

Project Mirror Labyrinthの戦闘は、ターン制のカードバトルです。手札からカードを使用するためにアクションポイント（ATP）を消費します。徐々に難易度が上がるステージを生き抜くためには、攻撃的なプレッシャーと防御的な軽減の健全なバランスを維持する必要があります。

### カードタイプの理解
カードはいくつかの異なるカテゴリーに分類され、それぞれ戦術において特定の役割を担っています。

| カードカテゴリー | コア機能 | 戦術的なユースケース |
| :--- | :--- | :--- |
| **攻撃カード** | ターゲットに物理または属性ダメージを与える | 敵のHPを減らし、混乱（姿勢）を崩す |
| **防御カード** | ブロック、シールド、バリア値を生成する | 敵のターン中の大ダメージを軽減する |
| **ユーティリティカード** | カードを引く、ATPを生成する、デバフを付与する | デッキの回転を速め、ボスのギミックを弱体化させる |

### 姿勢（混乱ゲージ）とATPの管理
キャラクターの戦闘効率は、主に「Posture（姿勢/混乱ゲージ）」と「ATP（アクションポイント）」という2つの資源に依存しています。
- **Posture（姿勢）**: 姿勢は第二の防御バーとして機能します。姿勢が崩れる（ブレイクする）と、気絶（スタン）状態になり、莫大なダメージを受けます。ターンごとの姿勢回復力を得るために、早い段階でパッシブスキル「**Breathing（呼吸）**」を獲得しておきましょう。
- **ATP**: カードを使用するためにATPを消費します。ATPの最大値は、**Condense（凝縮）**を使用し、Outfitをアップグレードすることで拡張されます。

<Section icon={Trophy} title="育成システム：抽出、衣装＆通貨">

Project Mirror Labyrinthにおける長期的な強さは、独自のステータス、カードセット、パッシブ能力を持つキャラクタークラスとして機能する「Outfits（衣装）」によって決まります。これらのOutfitは、ガチャスタイルの「Reflection（抽出）」システムを通じて獲得します。

### 有効な引き換えコード（2026年）
アカウントの進行を急加速させるため、ゲーム内メニューでこれらのアクティブなコードを今すぐ引き換えましょう。

| プロモコード | 報酬アイテム | 最適な使い道 |
| :--- | :--- | :--- |
| **UrbanPlague** | **405 Fragments** | 特定の抽出（Targeted Reflection）バナーのために貯めておく |
| **OfficeDelay** | **1,620 Fragments** | 最高ティアのOutfitを引くための高価値ブースト |
| **NepoOffice** | **405 Fragments** | 全般的なアカウントの進行とカードのアップグレード |
| **FauvistStudents** | **405 Fragments** | 高ティアのパッシブを解放するために貯めておく |
| **Welcome** | **350 Fragments, 500 Condense, 3 Seeds of Light** | 初期のOutfitデフラグ用のスターターパック |

### 育成ループ（プログレッションループ）
基本キャラクターを強力な戦力へと育てるには、以下の標準的な育成ループに従ってください：
1. **Fragmentsの獲得**: ダンジョン挑戦をクリアし、アクティブコードを引き換えてFragmentsを蓄積します。
2. **Reflectionsの抽出**: Fragmentsを消費して、「Targeted Reflections（特定抽出）」（フィリップやクローマーなどの特定の環境トップOutfit向け）または「Standard Reflections（常設抽出）」を引きます。
3. **Condenseの周回**: Library（図書館）ステージを周回して**Condense**を収集します。
4. **Outfitのデフラグ**: Condense、Seeds of Light、Fragmentsを使用してOutfitをデフラグし、隠されたパッシブ能力を解放して最大ATPを増加させます。
