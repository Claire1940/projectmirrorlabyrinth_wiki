<Section icon={Key} title="有効なコードと報酬の受け取り方法">

**Project Mirror Labyrinth Library**での進行を早めるためには、有効なコードの引き換えが不可欠です。これらの報酬からは、冒険の序盤で強力なOutfit（衣装）を解放するために必要なFragmentsや素材を入手できます。ほとんどのコードは、ゲーム内メニューで有効化する前に、公式のRobloxコミュニティに参加し、ゲームに「いいね！」を押しておく必要があります。

| コード | 報酬 | タイプ |
| :--- | :--- | :--- |
| **OfficeDelay** | 1,620 Fragments | 高価値通貨 |
| **UrbanPlague** | 405 Fragments | 育成素材 |
| **NepoOffice** | 405 Fragments | 育成素材 |
| **FauvistStudents** | 405 Fragments | 育成素材 |
| **Welcome** | 350 Fragments, 500 Condense, 3 Seeds of Light | スターターバンドル |

<Callout type="warning" title="大文字・小文字の区別">
Project Mirror Labyrinthのすべてのコードは**大文字と小文字を区別**します。エラーを避けるため、大文字を含めて表示されている通りに正確に入力してください。
</Callout>

### コードの引き換え方法
1. Robloxでゲームを起動します。
2. 画面左側にある**Menu**を開きます。
3. **Codes**アイコンを選択します。
4. テキストボックスにコードを入力または貼り付けます。
5. **Confirm**を押して報酬を受け取ります。

<Section icon={Swords} title="戦闘メカニクスとカードの種類">

迷宮での戦闘はターン制で、手札からカードを出して敵の行動に対処していきます。攻撃カードと防御カードのバランスを理解することが、ダンジョン攻略の成否を分けます。

| カードカテゴリ | 主な機能 | 戦略的な用途 |
| :--- | :--- | :--- |
| **攻撃カード (Attack)** | 敵に直接ダメージを与える | 敵が脆弱な状態のときやHPが低いときに優先して使用。 |
| **防御カード (Defense)** | 受けるダメージを軽減/ブロックする | エリートモンスターからの強力な攻撃を凌ぐために必須。 |
| **ユーティリティカード (Utility)** | ステータスを強化、またはカードをドローする | 次のターン以降で強力なコンボを仕掛けるために使用。 |
| **E.G.Oカード (EGO)** | 影響力の大きい特殊能力 | ボス戦や難しいウェーブを突破するために温存する。 |

<Section icon={Target} title="ダンジョンへの進入と探索ガイド">

**Project Mirror Labyrinth Library**のロビーは雰囲気重視でミニマルなデザインになっているため、最初の挑戦時は戸惑うかもしれません。迷宮へと降りるには、主に2つの方法があります。

<Steps>
  <Step num="1" title="TABメニューからのルート">
    キーボードの**TABキー**を押してメインインターフェースを開きます。そこから**Play**ボタンを選択すると、ダンジョンの開始エリアにテレポートします。
  </Step>
  <Step num="2" title="NO ENTRY（進入禁止）ゲート">
    没入感のあるスタートがお好みの場合は、ロビーを歩いて物理的な**NO ENTRY（進入禁止）ゲート**を調べます。これがローグライト体験への手動ポータルとなっています。
  </Step>
  <Step num="3" title="迷宮の探索">
    内部に入ったら、カードを使って部屋をクリアしていきます。道中では、デッキに新しいカードを追加したり、既存のカードをアップグレードしたりできるノードが出現することがあります。
  </Step>
  <Step num="4" title="メンテナンスへの対応">
    火曜日にダンジョンに入れなくなった場合は、時間を確認してください。ゲームは毎週**東部時間（EST）午後2:00頃**に定期メンテナンスを行います。
  </Step>
</Steps>

<Section icon={Sparkles} title="Reflection（鏡像）、Outfit（衣装）、そして育成">

ゲームの「ガチャ」要素は**Reflections（鏡像）**を通じて行われます。Fragmentsを消費して**Outfits（衣装）**を引くことができ、これらはキャラクターの「クラス」や「ビルド」として機能し、独自のステータスやパッシブスキルを提供します。

### 育成用の通貨

| 通貨 | 主な入手方法 | 主な用途 |
| :--- | :--- | :--- |
| **Fragments** | コード、ダンジョン攻略 | Reflection（鏡像）ガチャ、新スキルの習得。 |
| **Condense** | 特別報酬、マイルストーン | Outfit（衣装）の**Defragmenting（デフラグメント）**プロセスに使用。 |
| **Seeds of Light** | 高難易度の報酬 | 高度なパッシブ能力やアップグレードの解放。 |
| **Reflection Tickets** | イベント、コード | ピックアップ（Targeted）または通常（Standard）ガチャの無料チケット。 |

<CardGrid cols={3}>
  <Card title="ピックアップガチャ (Targeted Reflections)">
    特定のビルドを完成させるために**特定のOutfit（衣装）**を狙っているプレイヤーに最適です。ピックアップ対象の出現率がアップしています。
  </Card>
  <Card title="通常ガチャ (Standard Reflections)">

  </Card>
  <Card title="デフラグメント (Defragmentation)">
    強化システムです。Outfitを入手したら、CondenseとSeeds of Lightを使ってその潜在能力を最大限に引き出しましょう。
  </Card>
</CardGrid>
