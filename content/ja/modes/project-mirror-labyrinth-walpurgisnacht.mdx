<Section icon={Swords} title="新人格分析：南部親指マフィア">

**project mirror labyrinth ヴァルプルギスの夜** アップデートにおける最大の目玉は、間違いなく「南部親指マフィア」の人格です。このユニットは、洗練された「弾丸システム」と2つのステータス効果に焦点を当てた特徴を持っています。通常のアタッカーとは異なり、南部親指マフィアは32発という限られた弾丸リソースを消費して強力なスキルを発動します。

| スキル名 | 基本威力 | 主な効果 | 消費リソース |
| :--- | :--- | :--- | :--- |
| **ナイフ戦闘 (Knife Combat)** | 中 | 一般的な貫通ダメージ | 弾丸 0 |
| **トミーバースト (Tommy Burst)** | 高 | 多段ヒットによる破裂付与 | 弾丸 2-4 |
| **ガットジャブ (Gut Jab)** | 基本 8 | 負傷ステータスを付与 | 弾丸 0 |
| **アンロード (Unload)** | 極大 | 全弾丸を消費してバースト攻撃 | 残り全弾 |
| **回避 (Evade)** | 防御 | 回避成功時に弾丸を32発盗む | なし |

パッシブスキル**「二重処罰 (Double Punishment)」**は、このユニットのユーティリティの核となります。これは「破裂」と「負傷」のステータス効果の間に相互作用ループを生み出します。ターゲットに破裂があって負傷がない場合に攻撃が命中すると、負傷を1付与します。逆に、負傷があって破裂がない場合に攻撃が命中すると、破裂を3付与します。これにより、南部親指マフィアは「破裂の悪魔」と化し、1ターンの間に莫大なダメージ倍率を蓄積することができます。

<Callout type="success" title="メタのコツ">
「アンロード」スキルは、現在ゲーム内で最も強力なフィニッシャーの一つです。32発すべての弾丸を消費することで、次のターンに消費した弾丸の半分の値に等しい破裂の威力を付与し、確実なキルウィンドウを作り出します。
</Callout>

</Section>

<Section icon={Shield} title="戦闘メカニクス：「破裂」と「負傷」のシナジー">

**project mirror labyrinth ヴァルプルギスの夜** のコンテンツを攻略する上で、破裂と負傷の相互作用を理解することは不可欠です。出血はコイン判定に基づいてダメージを与えますが、破裂はターゲットが攻撃を受けるたびに固定ダメージを与えます。新しいヴァルプルギスの夜ユニットは、破裂付与の威力を高める「負傷」を導入することで、この仕組みをさらに強化しています。

| ステータス効果 | メカニクス | 戦略的価値 |
| :--- | :--- | :--- |
| **破裂 (Rupture)** | ヒットごとの固定ダメージ | 多段ヒットスキルに最適 |
| **負傷 (Wounded)** | 破裂獲得量を倍増 | ボス戦で必須 |
| **出血 (Bleed)** | スキル使用時にダメージ | 補助的なサポートダメージ |
| **オーバーヒート (Overheat)** | 弾丸使用によるデメリット | リソース管理の制限要素 |

ヴァルプルギスの夜の試練に挑むチームを編成する際は、「弾丸オーバーヒート」の数値を管理する必要があります。「アンロード」のようなスキルを使用すると、短期的には莫大なメリットが得られますが、「回避」スキルで弾丸プールを補充できない場合、ユニットが無防備になる危険があります。目標は、破裂の威力を20以上に到達させることです。これにより、鏡の迷宮に登場する最も手強いボスのHPゲージすらも効率よく削り取ることができます。

<Callout type="warning" title="リソース管理">
オーバーヒートの蓄積量には十分注意してください。南部親指マフィアは強力ですが、余剰の弾丸管理を怠ると、長期戦においてコイン威力が低下する原因になります。
</Callout>

</Section>

<Section icon={Target} title="イベント進行ステップバイステップガイド">

**project mirror labyrinth ヴァルプルギスの夜** イベントを進めるには、フラグメントの獲得とユニットの効率を最大化するための特定の進行ルートが必要です。限られたリソースを無駄にしないために、以下の手順に従ってください。

<Steps>
  <Step num="1" title="アカウントレベル上げ">
    通常の鏡ダンジョンを周回して、アカウントレベル60を目指しましょう。これはヴァルプルギスの夜ゲームモードに参加するための必須条件です。所持している場合は、経験値ブースターを使用してプロセスを加速させましょう。
  </Step>
  <Step num="2" title="フラグメント収集">
    最低でも400〜450個のフラグメントを集めることに集中してください。フラグメントはヴァルプルギスの夜ガチャの主要通貨であり、運悪く自力で引けなかった場合に、新しい人格を確定交換（天井）することができます。
  </Step>
  <Step num="3" title="ガチャの優先順位">
    ヴァルプルギスの夜限定ガチャにリソースを集中させましょう。これらのユニット（南部親指マフィア、ジャガーノート、プリースト）は、イベント終了後に恒常ガチャに追加されないため、最優先で確保すべきターゲットです。
  </Step>
  <Step num="4" title="同期化段階（Uptying）の強化">
    南部親指マフィアを獲得したら、すぐに同期化段階（Tier）3または4のパッシブを解放することを目指しましょう。「二重処罰」パッシブは、高いアップグレード段階に達して初めてその真価を発揮します。
  </Step>
</Steps>

<Callout type="tip" title="最適化">
レベル60の制限に苦戦している場合は、デイリーミッションのクリアと週に1度のダンジョンリセットの消化に集中してください。これらが最も効率よくアカウント経験値を獲得できる方法です。
</Callout>

</Section>
