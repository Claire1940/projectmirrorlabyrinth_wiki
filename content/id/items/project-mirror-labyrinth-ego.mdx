<Section icon={Map} title="Cara Memulai Run Dungeon Pertama Anda">

Memulai run di Project Mirror Labyrinth cukup sederhana, tetapi menavigasi antarmukanya bisa membingungkan bagi pemain baru. Game ini menawarkan dua metode utama untuk memasuki labirin dan mulai membangun deck Anda.

<Steps>
  <Step num="1" title="Luncurkan Game">
    Buka [Halaman Roblox Project Mirror Labyrinth](https://www.roblox.com/games/116276659864007/Project-Mirror-Labyrinth) resmi dan luncurkan pengalamannya. Pastikan Anda menggunakan keyboard dan mouse, karena dukungan kontroler tidak terdaftar secara resmi.
  </Step>
  <Step num="2" title="Bergabung dengan Komunitas">
    Pastikan untuk bergabung dengan [Komunitas Roblox Project Mirror Labyrinth](https://www.roblox.com/communities/127473071/Project-Mirror-Labyrinth) resmi. Ini diperlukan untuk berhasil menukarkan kode aktif.
  </Step>
  <Step num="3" title="Masuki Labirin">
    Tekan **TAB** pada keyboard Anda untuk membuka menu utama dan klik **Play**, atau berjalanlah ke dan berinteraksi dengan **gerbang NO ENTRY** yang terletak di lobi.
  </Step>
  <Step num="4" title="Tukarkan Kode Awal">
    Buka menu Codes di sisi kiri layar Anda dan masukkan kode yang berfungsi seperti **Welcome** atau **OfficeDelay** untuk mendapatkan Fragment dan material awal.
  </Step>
</Steps>
