<Section icon={Swords} title="Mekanik Pertarungan dan Strategi Membangun Dek">

Pertarungan di Project Mirror Labyrinth berbasis giliran dan berputar di sekitar memainkan kartu dari tangan Anda untuk membalas tindakan musuh. Seiring kemajuan Anda melalui dungeon, Anda akan menambahkan kartu baru ke dek Anda, menciptakan sinergi yang dapat menentukan keberhasilan atau kegagalan run Anda. Memahami perbedaan antara jenis kartu adalah langkah pertama untuk menguasai meta.

    - **Terbaik untuk**: Farming cepat dan lantai dungeon awal.

    - **Terbaik untuk**: Pertemuan bos dan labirin tingkat kesulitan tinggi.

  <Card title="Hibrida Utilitas">

| Kategori Kartu | Jenis Efek | Nilai Strategis |

| **Serangan** | Damage Langsung | **Tinggi**: Diperlukan untuk mengakhiri pertarungan dengan cepat. |

<Section icon={Shield} title="Eksplorasi Dungeon dan Pemeliharaan">

Inti dari **panduan project mirror labyrinth** melibatkan navigasi dungeon yang seperti labirin. Ini dibuat secara prosedural, artinya jalur yang Anda ambil akan berubah setiap kali Anda bermain. Anda harus memilih di antara node yang berbeda, seperti Pertarungan, Pertemuan Elite, atau Tempat Istirahat.

<Steps>
  <Step num="1" title="Memulai Run">
    Buka menu dalam game dengan menekan **TAB** dan pilih "Play," atau berjalanlah ke gerbang **NO ENTRY** di lobi untuk memulai.
  </Step>
  <Step num="2" title="Pilih Rute Anda">
    Lihat peta dan rencanakan jalur Anda. Prioritaskan node yang menawarkan hadiah kartu di awal untuk menstabilkan dek Anda.
  </Step>
  <Step num="3" title="Kelola Pertemuan">
    Terlibat dalam pertarungan kartu. Menangkan pertempuran untuk mendapatkan Fragments dan pilihan kartu baru.
  </Step>
  <Step num="4" title="Mencapai Luxcavation">
    Navigasikan menuju lapisan yang lebih dalam untuk membuka hadiah tingkat tinggi dan pertarungan bos.
  </Step>
</Steps>

Penting untuk dicatat bahwa game ini menjalani pemeliharaan rutin untuk memastikan stabilitas dan menambahkan konten baru seperti pembaruan "Lei Heng" atau "Not Binah". Selama waktu ini, akses dungeon mungkin dibatasi.

| Acara | Waktu Umum (EST) | Dampak pada Pemain |
| :--- | :--- | :--- |
| **Pemeliharaan Mingguan** | Selasa 14:00 | Server offline, progres tidak tersimpan. |
| **Penyebaran Patch** | Tidak teratur / Akhir pekan | Kode baru dirilis, perubahan meta. |
| **Reset Harian** | 00:00 | Penyegaran misi, hadiah login. |

<Callout type="info" title="Info Sistem">
Jika Anda mengalami kesalahan "NO ENTRY" di luar jam pemeliharaan, cobalah bergabung kembali ke server baru. Pembaruan Roblox terkadang dapat menyebabkan instansi lobi tidak sinkron dengan backend game.
</Callout>

</Section>
