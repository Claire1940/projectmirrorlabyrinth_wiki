<Section icon={Target} title="Langkah demi Langkah: Cara Menjalankan Dungeon Pertama Anda">

Memasuki dungeon dan bertahan dalam beberapa pertempuran pertama Anda bisa jadi menakutkan. Ikuti alur langkah demi langkah ini untuk memulai run Anda dan membangun deck yang stabil.

<Steps>
  <Step num="1" title="Akses Gerbang Dungeon">
    Di lobi utama, Anda dapat memulai run dengan menekan **TAB** pada keyboard dan mengklik **Play**. Atau, berjalanlah ke dan berinteraksi dengan gerbang fisik **NO ENTRY** yang terletak di dunia game.
  </Step>
  <Step num="2" title="Draft Deck Awal Anda">
    Setelah masuk, Anda akan menerima satu set dasar kartu serangan dan pertahanan. Jangan abaikan opsi pertahanan; memblokir kerusakan yang masuk sangat penting untuk bertahan dalam run yang dalam.
  </Step>
  <Step num="3" title="Terlibat dalam Pertempuran Taktis">
    Pertempuran berbasis giliran. Analisis niat musuh yang ditampilkan di atas kepala mereka. Gunakan kartu pertahanan untuk memitigasi serangan berat dan gunakan kartu serangan Anda selama ada celah untuk mengurangi kesehatan mereka.
  </Step>
  <Step num="4" title="Kumpulkan dan Habiskan Fragments">
    Selesaikan pertempuran untuk mengumpulkan Fragments. Setelah kembali ke lobi, habiskan Fragments ini pada banner Reflection untuk melakukan roll guna mendapatkan Outfit yang lebih kuat.
  </Step>
</Steps>

<Callout type="info" title="Pemeliharaan Mingguan">
Pengembang melakukan pemeliharaan game terjadwal setiap hari **Selasa sekitar pukul 14:00**. Akses server, entri dungeon, dan penukaran kode mungkin dinonaktifkan sementara selama jendela waktu ini.
</Callout>

</Section>
